#
[ Criação de Clã ]
 1. Precisa ter em sua posse um Emperium para criar um clã. 
 (Pode ser obtido por Quest/Monstros.)
 
 2. Maneiras de criar um Clã: 
 /guild (guild) 
 'Pressione o atalho de Clã [Alt+G].
 
 3. ALT+G: Abre a janela de informações do clã.
#
 
[ Janela de informação do Clã ] 
#
 1. Nome do Clã, Experiência, Nível do Clã, Emblema,
  Líder do Clã, Inimigos, Pontos, Nível médio dos
  membros, Alianças, Territórios, Tendências.
#
 
 2. Informações dos Membros : 
  Contém as informações dos membros do clã e a
  possibilidade de retira-los do mesmo. Deve-se remover
  todos os membros, se o líder do clã pretende desfazê-lo.
#
 
 3. Posição : 
 Taxa máxima de doação é de 50%
#
 
 4. Habilidades do Clã : 
 Quando o Clã aumenta de nível, o líder pode usar
 os pontos para atribuir nas habilidades e investir
 nelas, uma vez que as mesmas estejam disponíveis.
#
 
 5. Histórico : 
 Lista de expulsos do clã
 Também está incluso a razão da exclusão...
#
 
 6. Aviso do Clã
 Notificação para os membros ao entrarem no jogo.
#
 
[ Emblema ]
#
 1) 24x24 pixel de 256-color bitmap (BMP)
 Formato da imagem para criar Emblemas.
#
 
 2) Copie o arquivo do emblema para a pasta "Emblem" 
 na pasta onde o seu Ragnarok Online foi instalado.
#
 
 3) Janela de informação de Emblema
 Clique "Editar" para mudar o emblema do clã.
#
 
 4) O Emblema do clã só pode ser 
 modificado pelo líder.
#
